このノート運用の始め方

質問をメモで終わらせず、あとで読み返せる小さなHTML記事に変える。

Created: 2026-06-05

Overview

このリポジトリでは、Codexに投げた疑問を1テーマ1ページの静的HTMLとして保存します。ビルドツールを使わないため、差分確認とGitHub Pages公開が単純です。

Flow

  1. AskスマホやPCから `質問:` で始まる依頼をCodexに送る。
  2. WriteCodexが公式情報を確認し、HTMLノートを作る。
  3. Review差分と参照URLを確認する。
  4. Publish問題なければcommitしてGitHub Pagesへ反映する。

Prompt Example

質問: Web ComponentsとReactコンポーネントの違いを、初心者向けに図解つきでノート化して

Summary

最初に必要なのは、静的HTMLサイト、Codex向けのルール、公開先の3つです。このリポジトリはその最小構成を用意しています。
