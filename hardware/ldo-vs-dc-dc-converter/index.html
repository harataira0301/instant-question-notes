LDOとDC-DCコンバータの違い

静かで単純なLDO、効率重視のDC-DC。選び方は「電圧差、電流、熱、ノイズ」で決まる。

Created: 2026-06-07

Overview

LDOは低ドロップアウトのリニアレギュレータです。入力と出力の差分を熱として捨てながら、出力電圧を安定させます。回路は単純でノイズが少ない一方、電圧差と電流が大きいほど発熱します。

DC-DCコンバータは、スイッチ、インダクタ、コンデンサなどを使って電力を変換します。降圧、昇圧、反転などに対応でき、効率は高くしやすい一方、スイッチングノイズや基板レイアウトへの配慮が必要です。

Comparison

観点 LDO DC-DCコンバータ
基本方式 余った電圧を熱として消費するリニア方式。 スイッチングでエネルギーを移し替える方式。
効率 おおよそ Vout / Vin に近い。電圧差が大きいと悪化する。 条件が合えば高効率にしやすい。損失はIC、インダクタ、スイッチング条件に依存する。
発熱 (Vin - Vout) x Iout が主な熱になる。 LDOより発熱を抑えやすいが、大電流では部品選定と放熱設計が必要。
ノイズ 低ノイズにしやすく、アナログ/RF/ADC周辺に向く。 スイッチングリップルやEMIが出るため、レイアウトとフィルタが重要。
部品点数 少ない。入力/出力コンデンサ程度で済むことが多い。 インダクタ、ダイオードまたは同期スイッチ、補償部品などが必要な場合がある。
電圧変換 基本は降圧のみ。入力は出力より少し高い必要がある。 降圧、昇圧、昇降圧、反転などを構成できる。
向く用途 小電流、低ノイズ、電圧差が小さい電源、DC-DC後段のポストレギュレータ。 大きな電圧差、大電流、電池寿命重視、発熱を抑えたい電源。

効率と熱の見積もり

LDOは考え方が単純です。5 Vから3.3 Vを作り、100 mA流す場合、LDOで消費される熱は次のようになります。

損失 = (5.0 V - 3.3 V) x 0.1 A = 0.17 W
効率の目安 = 3.3 V / 5.0 V = 66%

100 mAなら扱いやすいことが多いですが、1 Aになると1.7 Wの熱になります。小さなICではかなり厳しくなります。このように、LDOは「電圧差 x 電流」を最初に見るのが大事です。

選び方

  1. 電圧差を見るVinとVoutが近いならLDOが候補。差が大きいならDC-DCを優先する。
  2. 電流を見る小電流ならLDOで十分なことが多い。大電流なら熱の見積もりを先に行う。
  3. ノイズを見るADC、センサ、RFなどノイズに弱い部分はLDO、またはDC-DC後段にLDOを置く。
  4. 実装難度を見るDC-DCは部品配置、GND、スイッチング電流ループ、インダクタ選定が効く。

よくある構成

実際の基板では、どちらか一方だけでなく、DC-DCで大きく降圧してからLDOでノイズを落とす構成がよく使われます。

Battery / USB / Adapter
  -> DC-DC converter: 効率よく中間電圧へ変換
  -> LDO: 低ノイズな3.3 Vや1.8 Vを作る
  -> MCU / Sensor / ADC / RF

Summary

LDOは単純で低ノイズですが、電圧差と電流が大きいと熱が問題になります。DC-DCコンバータは効率と変換自由度に優れますが、スイッチングノイズとレイアウト設計が重要です。迷ったら、まずLDOの損失を計算し、熱が厳しければDC-DCを検討します。
