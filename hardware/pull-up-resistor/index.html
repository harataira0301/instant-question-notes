プルアップ抵抗とは何か

入力ピンや信号線を「何もしていない時でも確実にHIGHへ寄せる」ための抵抗。

Created: 2026-06-07

Overview

プルアップ抵抗は、信号線を電源側に弱くつなぐ抵抗です。スイッチが開いている時や、ICの出力が信号線を駆動していない時に、入力がふらふらした不定状態になるのを防ぎます。

逆に、信号線をGND側へ弱くつなぐ抵抗はプルダウン抵抗です。どちらも目的は「未接続っぽい状態に、あらかじめ意味のある電圧を与えること」です。

Basic Circuit

なぜ必要か

デジタル入力は、HIGHかLOWのどちらかとして読みたいものです。しかしスイッチが開いている時の入力ピンは、どこにも強く接続されていないことがあります。この状態はフローティングと呼ばれ、周囲のノイズや手の接近だけで値が変わることがあります。

プルアップ抵抗を入れると、通常時は入力ピンがVCC側に引かれてHIGHになります。ボタンを押した時だけGNDへ強くつながり、LOWとして読めます。

抵抗値の考え方

抵抗値は小さすぎると、ボタンを押した時や信号線をLOWにした時に無駄な電流が増えます。大きすぎると、線の容量やノイズの影響でHIGHに戻るのが遅くなったり、不安定になったりします。

ボタン入力では、まず10 kΩ前後がよく使われます。I2Cのような通信線では、バス容量、通信速度、ICがLOWに引き込める電流から決める必要があります。

例: 3.3 Vで10 kΩのプルアップ
LOW時に流れる電流 = 3.3 V / 10,000 Ω = 0.33 mA

I2Cでの役割

I2CのSDA/SCLは、複数のICが同じ線を共有します。そのため、各ICがHIGHを強く出し合うのではなく、LOWだけを引き込み、HIGHはプルアップ抵抗に任せる構成が使われます。

この方式では、誰かがLOWにすればバスはLOWになります。誰もLOWにしていない時だけ、プルアップ抵抗でHIGHへ戻ります。複数デバイスが同じ線を安全に共有するための重要な仕組みです。

よくある失敗

  • 入力ピンをプルアップもプルダウンもせず、値がランダムに見える。
  • 外付けプルアップとマイコン内蔵プルアップの両方を意識せず使い、合成抵抗値が想定とズレる。
  • I2Cモジュールを複数つなぎ、各基板のプルアップが並列になって強すぎる。
  • 長い配線や高速通信なのに抵抗値が大きすぎて、信号の立ち上がりが遅くなる。

Summary

プルアップ抵抗は、信号線の何もしていない状態をHIGHに決めるための部品です。ボタン入力ではフローティング防止、I2Cでは共有バスを安全にHIGHへ戻す役割があります。抵抗値は「電流を流しすぎないこと」と「信号が遅くなりすぎないこと」のバランスで決めます。
