ADCとナイキスト周波数

ADCはアナログ信号を一定間隔で読む。読める周波数の上限の目安が、サンプリング周波数の半分。

Created: 2026-06-09

Overview

ADCはAnalog-to-Digital Converterの略で、連続的に変化する電圧を、コンピュータやマイコンが扱える数値に変換する部品です。温度センサ、音声、電流、電池電圧などのアナログ量を読む時に使います。

ADCは信号を連続的に見続けるのではなく、一定間隔で「今の電圧はいくつか」を読みます。この読み取る速さをサンプリング周波数と呼びます。ナイキスト周波数は、そのサンプリング周波数の半分です。

サンプリング周波数 fs = 10 kHz
ナイキスト周波数 fN = fs / 2 = 5 kHz

サンプリングのイメージ

ナイキスト周波数とは

ナイキスト周波数は、サンプリング周波数の半分です。サンプリング周波数をfsとすると、ナイキスト周波数fNは次のようになります。

fN = fs / 2

たとえば、ADCを1秒あたり48,000回サンプリングするなら、ナイキスト周波数は24,000 Hzです。理論上、24,000 Hzより低い周波数成分だけでできた信号なら、十分な条件のもとで元の信号を復元できます。

ここで大事なのは「サンプリング周波数の半分までなら何でも完璧に測れる」という意味ではないことです。実際にはADCの入力帯域、分解能、ノイズ、フィルタ、ジッタ、信号源インピーダンスなども効きます。

エイリアシング

ナイキスト周波数を超える成分がADCに入ると、低い周波数の別の信号のように見えることがあります。これをエイリアシングと呼びます。いったんデジタル値になってからでは、本物の低周波なのか、高周波が折り返して見えているのかを区別できません。

具体例

サンプリング周波数が10 kHzのADCでは、ナイキスト周波数は5 kHzです。この時、7 kHzのノイズがADCに入ると、サンプリング後には3 kHzの成分のように見える場合があります。

fs = 10 kHz
fN = 5 kHz
入力に 7 kHz 成分がある

折り返しの例:
10 kHz - 7 kHz = 3 kHz

結果:
7 kHzの高周波が、3 kHzの信号のように見える

この3 kHzはデジタル処理だけでは取り除きにくいです。デジタル化された後は、普通の3 kHz信号と区別できないためです。

アンチエイリアスフィルタ

エイリアシングを防ぐには、ADCに入る前のアナログ段で、ナイキスト周波数より高い成分を十分に減らす必要があります。このために入れるローパスフィルタをアンチエイリアスフィルタと呼びます。

理想的には、欲しい周波数だけを通し、ナイキスト周波数以上を完全に切りたいところです。しかし現実のフィルタには遷移帯域があります。つまり、ある周波数から急にゼロにはならず、だんだん減衰します。そのため実務では、サンプリング周波数を必要最小限の2倍ぴったりではなく、余裕を持って高くすることが多いです。

選び方の目安

  1. 測りたい最高周波数を決める温度のようにゆっくり変わる信号か、音声や振動のように速く変わる信号かを見る。
  2. サンプリング周波数を決める最低でも測りたい最高周波数の2倍より高くする。実務ではフィルタ余裕を見てさらに高くする。
  3. アンチエイリアスフィルタを入れるADCの前で、不要な高周波やノイズを落とす。
  4. ADCの性能も確認する分解能、入力帯域、サンプリング速度、入力インピーダンス、ノイズを確認する。
  5. 必要ならオーバーサンプリングする高めのサンプリング周波数で取り、デジタルフィルタ後に間引くと設計しやすい場合がある。

よくある誤解

  • 「ナイキスト周波数までなら必ず正確に測れる」わけではない。ADCの精度やアナログ回路も効く。
  • 「サンプリング周波数を2倍にすれば十分」とは限らない。現実のフィルタには遷移帯域がある。
  • エイリアシングした成分は、デジタル化後に本物の低周波と区別しにくい。
  • 平均化すれば何でも良くなるわけではない。折り返した周期成分は平均化で消えないことがある。
  • ADCのサンプルレートだけでなく、入力回路、センサ、配線、ノイズ源も見る必要がある。

Summary

ADCはアナログ信号を一定間隔でサンプリングして数値に変換します。ナイキスト周波数はサンプリング周波数の半分で、これを超える周波数成分がADCに入ると、低い周波数へ折り返して見えるエイリアシングが起きます。実用設計では、測りたい信号帯域より十分高いサンプリング周波数を選び、ADCの前にアンチエイリアスフィルタを入れて、不要な高周波をデジタル化前に落とします。
